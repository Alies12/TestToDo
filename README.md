Уставнока и запуск
1.Установка зависимостей: npm install
2.Запуск: npm run dev

Фунционал приложения

1.Добавить задачи можно написав их в поле и после нажав кнопку "Добавить" или клавишу "Enter"
2.Редактирование задачи происходит двойным нажатием на неё
3.Удаление задач происходит нажатием на кнопку "Удалить" напротив задачи
4.Фильтрация задач происходит нажатием на одну из 3 кнопок снизу
5.Отметка выполненной задачи происходит нажатием на квадрат слева от неё
